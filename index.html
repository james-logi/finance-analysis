🚀 로컬 프록시 서버 설정 가이드

⚠ 왜 이 단계가 필요한가요? 브라우저는 보안상 외부 사이트(DART, SEC)에 직접 접근하는 것을 제한하고 있습니다. 안전하고 빠르게 사용하려면 사용자 PC에서 작은 프록시 서버를 띄워야 합니다.
※ 브라우저는 보안상 사용자 PC에서 임의의 명령(python 등)을 자동 실행할 수 없습니다. 한 번만 직접 실행해 주세요.
1단계 · Python 프록시 파일 다운로드
파일을 적당한 폴더에 저장하세요 (예: 바탕화면)
2단계 · 터미널에서 실행
python dart_proxy.py
python3 dart_proxy.py
▸ Windows 자세한 절차
① Win + R 키 → cmd 입력 → Enter (명령 프롬프트 열림)
② cd 명령으로 dart_proxy.py가 있는 폴더로 이동 (예: cd Desktop)
③ python dart_proxy.py 입력 후 Enter
④ "Proxy running on..." 메시지가 보이면 성공! 창을 닫지 말고 두세요.
▸ Mac 자세한 절차
① Cmd + Space → terminal 입력 → Enter
② cd 명령으로 dart_proxy.py가 있는 폴더로 이동 (예: cd ~/Desktop)
③ python3 dart_proxy.py 입력 후 Enter
④ "Proxy running on..." 메시지가 보이면 성공!
▸ Python이 설치되어 있는지 확인
터미널에서 python --version (Win) 또는 python3 --version (Mac/Linux) 입력.
"Python 3.x.x"가 보이면 이미 설치되어 있습니다.
없다면 python.org에서 설치하세요. (Win 설치 시 "Add Python to PATH" 체크 필수)
3단계 · 페이지에 연결
① 위 [로컬 프록시 사용] 체크박스 클릭
② [🔌 로컬 프록시 테스트] 버튼 클릭 → ✅ 성공 메시지 확인
③ 인증키 [✓ 확인] 후 분석 시작!
▸ Python 코드 직접 보기 (수동 저장용) 
from http.server import HTTPServer, BaseHTTPRequestHandler
from urllib.parse import unquote
import urllib.request

class Proxy(BaseHTTPRequestHandler):
    def do_GET(self):
        if self.path.startswith('/proxy?url='):
            target = unquote(self.path[len('/proxy?url='):])
            try:
                req = urllib.request.Request(target, headers={'User-Agent': 'Mozilla/5.0 FinScope test@example.com'})
                with urllib.request.urlopen(req, timeout=30) as r:
                    body = r.read()
                    ctype = r.headers.get('Content-Type', 'application/octet-stream')
                self.send_response(200)
                self.send_header('Content-Type', ctype)
                self.send_header('Access-Control-Allow-Origin', '*')
                self.end_headers()
                self.wfile.write(body)
            except Exception as e:
                self.send_response(500)
                self.send_header('Access-Control-Allow-Origin', '*')
                self.end_headers()
                self.wfile.write(str(e).encode())
        elif self.path == '/ping':
            self.send_response(200)
            self.send_header('Content-Type', 'text/plain')
            self.send_header('Access-Control-Allow-Origin', '*')
            self.end_headers()
            self.wfile.write(b'pong')
        else:
            self.send_response(404)
            self.send_header('Access-Control-Allow-Origin', '*')
            self.end_headers()
    def do_OPTIONS(self):
        self.send_response(200)
        self.send_header('Access-Control-Allow-Origin', '*')
        self.send_header('Access-Control-Allow-Methods', 'GET')
        self.send_header('Access-Control-Allow-Headers', '*')
        self.end_headers()
    def log_message(self, *a): pass

if __name__ == '__main__':
    print('=' * 60)
    print('FinScope Proxy running on http://localhost:8765')
    print('Test: open http://localhost:8765/ping in browser')
    print('Stop: press Ctrl+C')
    print('=' * 60)
    HTTPServer(('localhost', 8765), Proxy).serve_forever()
